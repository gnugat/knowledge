# Kentel Mann / Leçon Zéro

Se présenter et dire d'où on vient (Oulpan – Brezhoneg 1, dialogue 1, p.34).

## Saludiñ / Saluer

| Brezhoneg          | Français                   |
|--------------------|----------------------------|
| Demat d'an holl    | Bonjour à tous             |
| Salud dit / deoc’h | Salut à toi / vous         |
| Mont a ra ?        | Ça va ?                    |
| Mat ar jeu ?       | Bien les affaires ?        |
| Mat an traoù ?     | Bien les choses ?          |
| Mont a ra mat      | Ça va bien                 |
| Mat-tre            | Très bien                  |
| Disteñget          | Distingué / Pas mal        |
| N'eo ket strikt    | C'est pas grave / tant pis |
| Kenavo             | Au revoir                  |
| Ken arc’hoazh      | À demain                   |
| Ken dilun          | À lundi                    |
| Devezh mat         | Bonne journée              |
| Noz vat            | Bonne nuit                 |
| Nozvezh vat        | Bonne soirée               |

> **Notenn / Note** :
>
> - _`Demat` : `Deiz` pour `jour` et `mat` pour `bon` / `bien`
>   Le suffixe `-vezh` marque la durée: `Deiz` / `Jour`, `Devezh` / `Journée`
>   `Noz` / `Nuit`, `Nozvezh` / `Soirée`, `Bloaz` / `An`, `Bloavezh` / `Année`_
> - _`Mont a ra` : `Mont` (aller, infinitif) + `a ra` (3ème pers. sing.)_
> - _`Traoù` : pl. de `tra` (chose).
>   Le suffixe `-où` est la marque du pluriel de beaucoup de noms communs_
> - _`Strikt`: `Strict`, et par extension `Grave`_

## Se présenter

| Brezhoneg             | Français                 |
|-----------------------|--------------------------|
| Peseurt anv 'peus ?   | Quel prénom as-tu ?      |
| Piv out ?             | Qui es-tu ?              |
| Eus pelec'h out ?     | D'où es-tu ?             |
| Me eo Yann, eus Brest | Moi c'est Yann, de Brest |

> **Notenn / Note** :
> 
> - _`Peseurt anv 'peus ?` utilise le verbe **avoir**, on ne peut pas répondre avec le verbe **être**.
>     On répond uniquement par son prénom : `Peseurt anv 'peus? Yann.`_
> - _`Eus` : `De` (origine, provenance), se prononce `Deus`_
> - _`Anv` : `Nom`, **V** en fin de mot se prononce **O** (`anv`, `atav`, `brav`...)_
> - _`Piv` : `Qui`, **IV** en fin de mot se prononce **OU** (`piv`, `liv`...)_
