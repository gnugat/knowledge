# Kentel Unan / Leçon Une

An anvioù-gwan / Les adjectifs (Oulpan – Brezhoneg 1).

## Anvioù-gwan / Adjectifs

| Brezhoneg | Français   |
|-----------|------------|
| Mat       | Bon / Bien |
| Fall      | Mauvais    |
| Bras      | Grand      |
| Bihan     | Petit      |
| Hir       | Long       |
| Kourt     | Court      |
| Vil       | Moche      |
| Pinvidig  | Riche      |
| Paour     | Pauvre     |
| Skuizh    | Fatigué    |
| Klañv     | Malade     |
| Gwir      | Vrai       |

> **Notenn / Note** :
>
> - _`Bihan-Bihan` : la répétition de l'adjectif renforce le sens : tout petit_

## Nac'hadur / Négation

| Brezhoneg  | Français      |
|------------|---------------|
| Ya         | Oui           |
| Nann       | Non           |
| N'eo ket   | Ce n'est pas  |

> **Notenn / Note** :
>
> - _La négation encadre le verbe : `n'` + verbe + `ket` (comme le `ne…pas` français)_
> - _`Ya` répond à une question **affirmative**_
> - _`Nann` répond à une question **négative**_
> - _`N'` + verbe + `ket` **nie** une réponse affirmative.
>   Elle se double : `Brav eo an amzer ? N'eo ket, n'eo ket brav, fall eo`_

## Diviz / Dialogue

| Brezhoneg                             | Français                               |
|---------------------------------------|----------------------------------------|
| Penaos eo Padrig ?                    | Comment est Padrig ?                   |
| Paour eo                              | Il est pauvre                          |
| Pinvidig eo Padrig ?                  | Est-il riche, Padrig ?                 |
| N'eo ket, n'eo ket pinvidig, paour eo | Non, il n'est pas riche, il est pauvre |
| Paour eo Padrig ?                     | Est-il pauvre, Padrig ?                |
| Ya gwir eo                            | Oui c'est vrai                         |

> **Notenn / Note** :
>
> - _Le verbe n'est jamais en 1ière position (`Bras eo`, `N'eo ket bras`)_

## Deizioù ar sizhun / Jours de la semaine

| Brezhoneg  | Français  |
|------------|-----------|
| Dilun      | Lundi     |
| Dimeurzh   | Mardi     |
| Dimerc'her | Mercredi  |
| Diriaou    | Jeudi     |
| Digwener   | Vendredi  |
| Disadorn   | Samedi    |
| Disul      | Dimanche  |

> **Notenn / Note** :
>
> - _Avec `Di-`: pas d'article en français : `Dilun` = lundi (prochain ou dernier)_
> - _Sans `Di-`: avec article en français : `Ar lun` = le lundi (tous les lundis)_

### Rimadell / Comptine

> **E Brezhoneg**:
>
> Dilun dimeurzh dimerc’her
> Ha yaou 'ta, ha yaou 'ta, ha yaou 'ta, ha yaou 'ta
> Dilun dimeurzh dimerc’her
> Diriaou ha digwener.
> Ha disadorn ha disul
> Ha yaou 'ta, ha yaou 'ta, ha yaou 'ta, ha yaou 'ta
> Ha disadorn ha disul
> Setu echu eo ar sizhun

> **En français**
> 
> Lundi mardi mercredi
> Bah oui, bah oui, bah oui, bah oui
> Lundi mardi mercredi
> Jeudi et vendredi
> Et samedi et dimanche
> Bah oui, bah oui, bah oui, bah oui
> Et samedi et dimanche
> Voilà la semaine est terminée
